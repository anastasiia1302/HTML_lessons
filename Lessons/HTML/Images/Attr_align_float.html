Язык HTML был разроботан британским ученым Тимом Бернерсом-Ли приблизительно в 986-1991 годах в стеах Европйского Центра ядерных иследований в Женеве (Швейцария). HTML создавался как язык для обмена научной и технической документацией, пригодный для использования людьми, не являющимися специалистами в области вёрстки. HTML успешно справлялся с проблемой сложости SGML путём определения небольшого набора структурных и семантичесих элементов - дескрипторов. Дескрипторы тоже часто называют "тегами". С помощью HTML можно легко создать относительно простой, но красиво оформленый документ. Помимо упрощения структуры документа, в HTML внесена поддержка гипертекста. Мультимидийные возможности были добавлены позже. Изначально язык HTML был задуман и создан как средство структуирования и форматирования документов без их привязки к средствам воспроизведения (отображения).

В идеале, текст с разметкой HTML должен был без стилистических и структурных искажений воспроизводиться на оборудовании с различной технической оснащённостью (цветной экран современного компьютера, монохромный экран органайзера, ограниченный по размерам экран мобильного телефона или устройства и программы голосового воиспроизведения текстов).
